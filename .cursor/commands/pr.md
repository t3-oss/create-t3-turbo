# Pull Request Generator

## Overview

Generate a comprehensive pull request description based on the diff between the current branch and its base branch, then either create a new PR or update an existing one. Uses the repository's PR template (`.github/PULL_REQUEST_TEMPLATE.md`) to maintain consistent structure.
